Project - Version Management

Manage the versions of your projects

Handle your releases and versions with this module. Create a new version for your project that groups all the tasks that will be in the next release.

This module comes with 5 version state:

  • Draft
  • Work In Progress
  • Staging
  • Production
  • Past

User guide

How to use this module?

  1. Go in Search/Versions and create a 'Version'.
  2. If you want to add a new 'Version State', go in Configuration/Version States.
  3. Add tasks to the 'Version' you want.
  4. Advance the 'Version State' when needed.

Particularities

  1. The 'Version States' which have their 'Done State' checkbox checked can only be activated if all their tasks are on 'Done' Stage.
  2. When a version of a Project is set on the 'Production' version state the previous 'Production' version of this project is automatically set on 'Past' version state.
